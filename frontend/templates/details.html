Introduction

Prediction of Soybean Yield earlier to harvest would help policymakers and farmers in taking proper marketing and storage measures. This project will enable farmers in determining the yield of their crop prior to cultivating the agricultural land, allowing them to make better decisions.Early crop yield prediction encourages the best crop choices and thereby increases food security.Crop yield prediction became difficult due to the interdependence of multiple factors such as crop genotype, environmental factors, management practices, and their interactions, but tdeep learning appears to have great potential in terms of increased accuracy and efficiency.In this research a deep learning framework, CNN is proposed for efficient soybean yield prediction.

Proposed System

The proposed system is Soybean Yield Prediction using CNN.
The major contributions are as follows.

1. The dataset consists of soil data,weather data,yield performance and management data .
2. The features of dataset are processed for dimensionality reduction
3. Classification model is trained to predict the soybean yield. CNN classifier is implemented.

Methodology

The proposed system is Soybean Yield Prediction using CNN.
It consists of six modules:
1.Data Collection : In this module data is collected from different sources.
2.Data Preprocessing : In this module data is preprocessed to remove unnecessary attributes.
3.Model Building : In this module CNN model is built to train and test the data.
3.Error caluculation :In this module error rate is calculated using performance metrics such as RMSE value,MSE value,MAE value,R^2 value
4.Yield Prediction : In this module average yield of soybean is predicted based on the available data.
5.Display Output : In this module the yireld predicted is displayed as a result.

Performance

A web application which predicts soybean yield at the earliest based on the available data.The performance is measured based on performance metrics such as RMSE value,MSE value,MAE value and R^2 value.This application predicts the soybean yield with high accuracy and less error rate.
